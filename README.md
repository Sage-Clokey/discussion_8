# BME 105 — Population Genetics Simulation

Wright-Fisher population genetics simulator built for BME 105 Discussion Section 8 at UC Santa Cruz.

## Contents
- **PopGen_Simulator.html** — Browser-based Wright-Fisher simulator with CSV batch input, Chart.js visualization, and auto-generated analysis
- **Population_Genetics_Analysis.html** — Static analysis report
- **simulations.csv** — 9-simulation batch (N=500/10000/∞ × p₀=0.1/0.5/0.9)
- **simulations_large.csv** — 54-simulation batch (6 population sizes × 9 starting frequencies)

## Key Concepts
- Genetic drift under the Wright-Fisher neutral model
- Hardy-Weinberg equilibrium
- Allele fixation and loss probability as a function of N and p₀
- Effect of population size on drift rate

## Usage
Open `PopGen_Simulator.html` in a browser, upload a CSV, and run simulations.
